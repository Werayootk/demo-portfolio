### Setup

1. Clone this repo
   ```sh
   git clone https://github.com/Your_username/React-Movie-TMDB.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Get API Key from
```
https://www.themoviedb.org/
```
4. In folder create new file `.env` for setup config to assign API_Key
   ```
   REACT_APP_API_KEY = API_KEY;
   ```
